## monospace git

Run given git command in each repository contained in this monospace

### Synopsis

Run given git command in each repository contained in this monospace (including root) concurrently. 

You can ommit execution in one or more projects using --project-filter[-out] flags.

Output mode is grouped by default, and git flags must be passed after a double hyphen.


```
monospace git <git-command> [git-args...] [-- git-flags...] [flags]
```

### Examples

```
  monospace git status -- --porcelain
# excluding root
	monospace -P root git status -- --ignored

```
